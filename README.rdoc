= Konsole DBus Development Framework

This is a framework for managing development Konsole windows using DBus.
Yes, we are so crazy!

== Commands

Currently, there are the following commands:

projects:: List available projects

  dev projects

change:: Change all the development windows to project

  dev change project

switch:: Change only current Konsole to project

  dev switch project

help:: Get help message
